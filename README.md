# FoodApp

A Java web-based food delivery application with restaurant browsing, cart, and order management.

## Features

- Browse restaurants and menus
- Add items to a session-based cart
- Place orders
- View order history
- User login and session management

## Tech Stack

- Java 17
- Jakarta Servlet API
- MySQL
- Apache Tomcat 10.1
- Eclipse IDE

## Getting Started

1. Clone the repository
2. Import the project into Eclipse as a Dynamic Web Project
3. Set up the MySQL database and update the connection details in the project
4. Deploy to Apache Tomcat 10.1
5. Run and access the app in your browser
